# Olist E-Commerce Analysis 🛒

## Project Overview
Analysed 99,000+ real Brazilian e-commerce orders 
using PostgreSQL and Excel to find key business insights.

## Tools Used
- PostgreSQL 17
- Excel (Dashboard)
- SQL

## Key Findings
- SP (Sao Paulo) generates 42% of total revenue
- Late deliveries cause 3x lower review scores
- Beauty & Health is the top revenue category
- Top seller generated 2.26 lakh in revenue

## Skills Demonstrated
- Data Cleaning
- SQL Joins, Group By, Subquery, Window Functions
- Dashboard Creation
- Business Insights
